/organisation get

get

get /organisation

  • Response

HTTP status code 200

Headers

  • Accept: (string)

    By default, all requests receive the v1 version. We encourage you to explicitly request this version via the Accept header.

    Example:

    application/vnd.jisc.api.v1+json
  • Cache-Control: required (string)

    Activates caching and defines cache behavior through cache response directives. Usually defines public or private (cacheable by proxy or not) and max-age for resource. See http://www.w3.org/Protocols/rfc2616/rfc2616-sec14.html for more information.

    Example:

    private, max-age=31536000
  • Access-Control-Allow-Origin: (*)
  • Access-Control-Expose-Headers: (X-RateLimit-Limit, X-RateLimit-Remaining, X-RateLimit-Reset)
  • X-RateLimit-Limit: required (integer - default: 1000)

    The maximum number of requests that the consumer is permitted to make per hour.

  • X-RateLimit-Remaining: required (integer)

    The number of requests remaining in the current rate limit window.

  • X-RateLimit-Reset: required (string)

    The time at which the current rate limit window resets in UTC epoch seconds

Body

Media type: application/json

Type: array of Organisation

Items: Organisation

  • organisationID: required (integer)
  • organisationName: required (string)
  • organisationType: required (one of charity, commercial, funder, furtherEducation, government, health, higherEducation, other, professionalBody, research, school, skills)

Example:

[
  {
    "organisationID": 274,
    "organisationName": "University of Winchester",
    "organisationType": "higherEducation"
  },
  {
    "organisationID": 1539,
    "organisationName": "Imperial College of Science, Technology and Medicine",
    "organisationType": "charity"
  }
]
